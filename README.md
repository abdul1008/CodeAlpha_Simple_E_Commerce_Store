This project is a Simple E-Commerce Website that allows users to browse products, view product details, add items to a shopping cart, and place orders online. The application focuses on providing a smooth and user-friendly shopping experience.

Users can register, log in, explore products by category, search for items, and manage their cart before checkout. The system automatically calculates the total price and generates an order summary.

An Admin Panel is included to manage products, categories, pricing, and orders. Admins can add, update, or delete products and monitor customer activity.

This project is built using modern web technologies and serves as a strong foundation for developing a complete online shopping platform.
